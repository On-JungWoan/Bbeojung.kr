광주광역시 버스 정보 제공 사이트

버스의 예상 승차인원 및 각종 정보를 제공합니다. 지역, 정류장, ARS_ID(또는 정류장명)을 입력하세요.

How it Works

특정 버스의 예상 승차인원을 예측하고,
사용자에게 서비스 하기까지의 과정을 소개합니다.

01

Pickle File Import

Python에서 사전에 미리 학습시킨
ML Model의 Pickle File을 Import 합니다.

02

Calculation

Django 내부 View 함수에서 연산을 진행하여
Predict Value를 산출합니다.

03

Show

Django Shorcut의 Render 메소드를 사용하여
html로 예측값을 넘겨줍니다.
